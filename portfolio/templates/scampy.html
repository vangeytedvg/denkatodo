SCAMPY

How it works

SCAMPY Got it's name when I was thinking of a name when I started the project. I was going to make this application a friend of mine. He never realy used a computer let it be scanning and email software. He told me he needed a simple program to create an email, use his gmail account for this, and that he needed to scan documents. The program he was thinking of would do this in one shot. So I came up with Scan And Mail with PYthon. This tool only sends email, but lets the user scan documents related to a specific email and then send them via Gmail. The screenshots are in Dutch because my friend only speaks Dutch. In the future this program will integrate the QT Translation system.
