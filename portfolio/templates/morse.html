How it works

As a said in the introduction, I always wanted to learn morse code. So, I wrote a Python program for this purpose.

While having a lot of fun making the program, I actually learned Morse-code!

The interface of the application is made with Qt-Designer. And is then compiled into python code

Here are some screenshots

The main screen

As you can see, the interface is quite simple. The user can either use real morse sounds or the so-called Dit Dah sounds. That means that a dot '.' is pronounced as 'Dit' and a dash '-' is pronounced as 'Dah'

The transmission speed can also be set. This tries to approach the WPM (Words Per Minute) standard.

Lightbulb mode

Morse code not consists of sounds, one can also learn to read light signals transmitting morse. On boats this is used often to communicate in silence.

The morse trainer can simulate this by using a flashing panel in the interface.
